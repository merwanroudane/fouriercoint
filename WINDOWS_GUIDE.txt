DOI: 10.1007/s00181-015-1028-6

## Package Status

✅ **Core Implementation Complete**
- Fourier cointegration test (OLS and DOLS)
- Critical values from Table 1 of the paper
- Automatic frequency selection
- F-test for Fourier significance
- Long-run variance estimation (Bartlett and QS kernels)
- Full documentation and examples

⚠ **Known Issues**
- Some numerical sensitivity with very strong cointegration and small noise
- DOLS with automatic bandwidth selection may need tuning for small samples
- Recommended to use with T >= 100 observations

## Quick Start

```python
import numpy as np
from fouriercoint import fourier_cointegration_test

# Generate sample data
np.random.seed(42)
T = 200

# I(1) process
x = np.cumsum(np.random.randn(T, 1))

# Cointegrated with structural break
t = np.arange(T)
break_point = T // 2
level_shift = np.where(t < break_point, 0, 3)
y = 2 + 0.5 * x + level_shift.reshape(-1, 1) + np.random.randn(T, 1)

# Run test
results = fourier_cointegration_test(
    y=y,
    x=x,
    m=0,  # Level shifts only
    kmax=3,
    significance_level=0.05
)

print(f"Test Statistic: {results['test_statistic']:.4f}")
print(f"Cointegration: {not results['reject_null']}")
```

## Usage Examples

### Basic Test

```python
from fouriercoint import fourier_cointegration_test

# Your I(1) data
results = fourier_cointegration_test(y, x, m=1)
```

### Get Critical Values

```python
from fouriercoint import get_critical_values

cv = get_critical_values(m=1, k=1, p=1)
print(f"5% critical value: {cv[0.05]:.4f}")
```

### OLS vs DOLS

```python
from fouriercoint import (
    fourier_cointegration_ols,
    fourier_cointegration_dols
)

# OLS (for exogenous regressors)
ols_result = fourier_cointegration_ols(y, x, m=1, k=1)

# DOLS (for potentially endogenous regressors)
dols_result = fourier_cointegration_dols(y, x, m=1, k=1, q=3)
```

### Multivariate

```python
# Multiple regressors
x_multi = np.cumsum(np.random.randn(T, 3), axis=0)
results = fourier_cointegration_test(y, x_multi, m=1)
```

## Troubleshooting

### Issue: LRV too small
**Solution:** Use larger bandwidth or add more noise to data

### Issue: Test statistic too high
**Solution:** Check data is actually cointegrated, increase sample size

### Issue: Import errors
**Solution:** Reinstall package: `pip uninstall fouriercoint && pip install -e .`
